sweeten (fn, argc)
Sweeten a function. This effectively allows functions taking multiple arguments to be "called without parentheses." You probably don't need to use this.

Parameters:

  • fn A function taking a fixed number of non-optional arguments.
  • argc The number of arguments fn takes. If fn is a table method, argc should include the hidden self argument.

Returns:

    A wrapper function for fn. If the wrapper is called with less than argc arguments, it will return a continuation object that can be called or concatenated with any remaining arguments. Otherwise, it will tail-call fn and return the result.

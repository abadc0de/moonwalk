Load an API class.

Mostly just a wrapper for require .

Usage:

     local api = require "moonwalk"
 api.load "transaction"
 api.handle_request(...)
class (docstring)
Describe an API class.

Syntactic sugar to hook up short descriptions to funtion tables, and register them with moonwalk.

Parameters:

  • docstring short description of the module

Returns:

    function(obj)

Usage:

     local api = require "moonwalk"
 api.class "Customer Transactions" {
   post = api.operation [[ ...
   ]] .. function(customer, amount) -- ...
   end,
 }

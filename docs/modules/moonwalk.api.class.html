constructor (title, operations)
Instantiate an API class. This function is not meant to be called directly (see proto.constructor ).

Parameters:

  • title Short description of the module.
  • operations Table containing operations.

Usage:

 api.class "Customer Transactions" {
   post = api.operation [[ ...

   end,

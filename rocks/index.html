moonwalk - Zero-friction REST APIs for Lua.

Moonwalk is a Swagger server implementation for Lua.

Moonwalk is designed to work under various host environments. Currently includes support for CGI, Mongoose/Civetweb, LuaNode. Support can easily be added for other host environments.
